Forma de trabajo remoto con git:

PASO 1:

- crear una nueva carpeta con click derecho, seguido abrimos una nueva terminal
- usamos "git init"
- usamos "git clone [url repositorio sin los [] ]"

Seguidamente, una vez realizemos un cambio y git lo marque como un archivo de color amarillo o verde, usamos:
- git add . o git add ruta/nombre archivo
- git commit -m "Nombre del commit (comentario, el cual guia a la siguiente persona que obtenga tus cambios)"
- git remote add origin master
- git push origin master
